# flutter_arduino_ble
Mobile application for communication with Arduino Bluetooth modules likes HM-10

## Test
```
 $ flutter run
```

## Build (Android)
```
 $ flutter build apk --split-per-abi
```

## Install (Debug Version)
```
 $ flutter install
```
